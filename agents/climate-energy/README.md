# CLIMATE — Energy & Sustainability Agent

SilkWeb climate agent that calculates carbon footprints, audits energy efficiency, scores ESG, and analyzes renewable feasibility.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | /calculate/carbon | Calculate CO2e from business activities |
| POST | /audit/energy | Energy efficiency audit for buildings |
| POST | /score/esg | ESG score calculation |
| POST | /analyze/renewable | Solar/wind feasibility analysis |

## Port: 3020
