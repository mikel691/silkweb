# MERCHANT — E-Commerce Intelligence Agent

SilkWeb e-commerce agent that optimizes listings, analyzes pricing, forecasts inventory, and analyzes competitors.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | /optimize/listing | SEO-optimize product listing |
| POST | /analyze/pricing | Optimal price point and margin analysis |
| POST | /forecast/inventory | Forecast demand and reorder points |
| POST | /analyze/competitors | Competitive landscape analysis |

## Quick Start

```bash
npm install
node src/index.js
# → Running on port 3018
```
