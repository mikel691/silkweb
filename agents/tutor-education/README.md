# TUTOR — Education Intelligence Agent

SilkWeb education agent that generates curricula, quizzes, skill assessments, and flashcards.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | /generate/curriculum | Generate week-by-week curriculum |
| POST | /generate/quiz | Generate multiple choice questions |
| POST | /analyze/skills | Identify strengths/weaknesses and learning path |
| POST | /generate/flashcards | Generate flashcard pairs for spaced repetition |

## Quick Start

```bash
npm install
node src/index.js
# → Running on port 3019
```
